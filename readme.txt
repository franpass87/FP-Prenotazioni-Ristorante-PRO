== Description ==
Modular restaurant booking plugin with Flatpickr integration, email notifications and basic marketing tracking.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory or install via composer:

```
composer install fp/prenotazioni-ristorante-pro
```

2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= How do I display the booking form? =
Use the `[ristorante_booking_form]` shortcode in any page or post.
